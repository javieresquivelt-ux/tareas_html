HTML

HTML son las siglas de Hyper Text Markup Lenguage, que pueden traducirse como lenguaje de marcas o marcado de hipertexto

El lenguaje HTML se emplea para crear las págunas web. Es muy fácil ver el código HTML, de una página web, la opción exacta cambia de un navegador
a otro y también puede cambiar de una versión a otra de un mismo navegador, pero suelen tener un mismo nombre similar.

HTML se compone de etiquetas que se escriben entre los símbolos menor que y mayor que.

Historia de HTML

Los inicicios del lenguaje HTML se remontan al año 1990, cuando Tim Berners-Lee creó la primera página web.

Versiones de HTML

Tim Berners-Lee definió la primera versión de HTML en el año 1991.

En la actualidad, la última versión de HTML es HTML5.
